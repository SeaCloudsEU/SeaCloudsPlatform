Welcome SeaClouds Unified Dashboard

This website is a proof of concept of the SeaClouds Dashboard as result of the early prototype of the First Year of SeaClouds Platform. This current version offers a limited functionality of what expected from the finished product in the SeaClouds project, containing a first version of the Matchmaking process (in the Discoverer & Planner components), a Deployment at IaaS level (in the Deployer component), a Monitoring mechanism to check sensors/information of the deployed application (in the Monitor component), and a first version of the SLA management checking QoS violations (in the SLA Service component).



SeaClouds Dashboard is part of the ongoing European research project EC-FP7-ICT-610531 SeaClouds. ONLY FOR INTERNAL USE - NOT FOR RELEASE (December 2014).
